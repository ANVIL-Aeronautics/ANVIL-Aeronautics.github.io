## 2. 🚀 Project: AeroRover
지상과 공중을 융합한 차세대 하이브리드 모빌리티 시스템 (Hybrid Mobility System).

### ⚙️ Core Technologies
* **지상:** 험지 돌파를 위한 Caterpillar(무한궤도) 시스템
* **공중:** 고출력 Ducted Fan 기반 양력 확보 및 Jet Vane 자세 제어
